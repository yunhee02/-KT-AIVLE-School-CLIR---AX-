# 고속도로 무단투기 탐지 AX 솔루션

고속도로 CCTV 영상을 활용하여 무단투기 차량을 탐지하고,  
번호판 인식 및 쓰레기 분류를 통해 단속과 운영 개선까지 연결한 프로젝트입니다.

---

## 📌 주요 기능
- 차량 탐지 및 추적 기반 무단투기 행위 탐지
- 쓰레기 유형 분류
- 번호판 인식 (OCR)
- 탐지 결과 기반 관제 및 운영 활용 구조 설계

---

## 🛠 사용 기술
- Python
- Jupyter Notebook
- RT-DETR (객체 탐지)
- DeepSORT (객체 추적)
- YOLOv8 (쓰레기 분류, 번호판 탐지)
- EasyOCR (번호판 인식)

---
